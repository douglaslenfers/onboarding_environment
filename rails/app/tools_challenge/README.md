# Desafio

Aplicação de cadastro de produtos:
- O produto deve possuir os campos SKU, Nome, Descrição, Quantidade e
Preço
- Deve ser possível filtrar os produtos pelos campos sku, nome e descrição

## Uso

```bash
docker-compose run rails bash
```

```bash
cd tools_challenge/
```

```bash
bundle install
```

```bash
rails server
```
